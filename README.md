# Protocol 27

Stellar Protocol 27 (“Zipper”) — Key Details and Impact
Protocol 27, codenamed “Zipper,” is a major Stellar network upgrade scheduled for a mainnet validator vote on July 8, 2026, introducing native authentication delegation, multisig improvements, and a Soroban security fix coinalertnews.com+1.

What Protocol 27 Does
Native Authentication Delegation – Allows one Stellar account to authorize another to act on its behalf directly at the network level, enabling social recovery for wallets and simplifying multisig setups coinalertnews.com+1.

Security Patch – Fixes a replay vulnerability in Soroban smart contracts by implementing address‑bound credentials, preventing valid authorizations from being reused in different contexts coinalertnews.com+1.

Developer & User Benefits – Reduces transaction steps, lowers operational costs, and improves developer workflows for wallets, payment apps, and regulated finance products coinalertnews.com+1.

Transition to Protocol 28 – Prepares the network for contract‑based authentication in standard Stellar accounts coinalertnews.com+1.

Technical Highlights lumenquery.io
Soroban Resource Model Updates – Increases CPU instruction limits (100M → 150M), read/write byte limits, and event sizes, allowing larger contracts to execute.

State Archival Improvements – Cheaper TTL extensions, batch TTL operations, and new RPC queries for archival status.

Throughput Gains – Raises per‑ledger transaction capacity (classic: 100 → 150; Soroban: 20 → 30), ~50% more TPS at consensus level.

Why It Matters Now
Protocol 27 arrives amid rapid growth in Stellar’s stablecoin and tokenized real‑world asset (RWA) activity — over $3.3B in RWAs are already issued on the network Live Bitcoin News+1. Stablecoin use includes USDC in UN‑linked initiatives, MoneyGram and Stripe collaborations, and PayPal’s planned Stellar rollout Live Bitcoin News+1. This positions Stellar as a low‑cost, regulated cross‑border finance platform and strengthens its role in institutional digital asset settlement Live Bitcoin News+1.

Testnet: New protocol already live; mainnet deployment will require node and tool updates coinalertnews.com+1.

Impact: If approved, changes will execute immediately on testnet, with broader rollout following compatibility updates coinalertnews.com+1.

In short: Protocol 27 “Zipper” is both a technical milestone (authentication delegation, security fix, throughput gains) and a strategic move to align Stellar with the growing institutional and stablecoin ecosystem, making it a key upgrade for developers, businesses, and users in regulated finance Live Bitcoin News+2.
